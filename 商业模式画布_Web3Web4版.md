## 一、价值主张（Value Propositions）

### 1.1 核心价值

| 价值点 | 传统竞品 | 爻影创新 | Web3/Web4 增强 |
|--------|----------|----------|-----------------|
| **占卜体验** | 文字输入 + 静态结果 | 手势摇卦 + 音效 + 动画 | Web4：空间手势 + 3D 卦象 |
| **解卦质量** | 模板化解读 | GPT-4o 流式个性化报告 | Web3：社区验证 + 声誉机制 |
| **数据主权** | 平台拥有用户数据 | — | Web3：用户拥有数据 + 可携带 |
| **社交属性** | 无 | 卦象分享 + 社群讨论 | Web3：卦象 NFT + 社交图谱 |
| **可访问性** | 付费墙 | 免费基础 + 代币激励 | Web3：代币无许可全球支付 |

### 1.2 价值主张详细描述

**核心产品价值**：
- 「以手摇卦，以智解象」——传统六爻 + 现代 AI 的融合体验
- 新中式极简美学，区别于市面「土味」占卜 App
- 纯代码音效引擎，无音频文件的轻量化体验

**Web3 增值**：
- 卦象 NFT 化：每次占卜可 mint 为 NFT，形成个人命运时间轴
- 社区共治：代币持有者参与产品决策（DAO）
- 数据变现：用户可选择将数据匿名化后上链，获得代币奖励

**Web4 增值**：
- 空间计算体验：Apple Vision Pro 中「虚空摇卦」
- AI Agent 陪伴：7×24 私人命理顾问，越用越懂你
- 多模态交互：语音 + 手势 + 眼动，自然流畅

## 二、客户细分（Customer Segments）

### 2.1 用户分层

| 层级 | 用户画像 | 需求痛点 | 付费意愿 |
|------|----------|----------|----------|
| **C1 核心用户** | 25-35岁都市白领，本科+，对传统文化有兴趣 | 压力大、焦虑，需要情绪出口 + 决策辅助 | 高（¥10-50/月） |
| **C2 文化爱好者** | 国潮消费者，汉服/茶道/国学爱好者 | 文化认同，追求「正宗」体验 | 中（¥5-20/月） |
| **C3 海外华人** | 海外留学生/工作者，文化归属感需求 | 异国他乡，需要文化连接 + 心理安慰 | 高（$5-20/月） |
| **C4 好奇尝鲜者** | Z 世代，喜欢新鲜体验 | 社交货币，朋友圈分享 | 低（免费 + 偶尔付费） |
| **B1 企业客户** | 文创品牌、博物馆、文化机构 | 需要国风互动体验装置 | 高（¥5万-50万/项目） |
| **B2 开发者** | 需要音效/手势 SDK 的开发者 | 轻量化、无版权风险的解决方案 | 中（$20-100/月） |

### 2.2 用户画像详情

**典型用户 A：都市焦虑白领**
- 年龄：28-35 岁
- 职业：互联网/金融/咨询
- 痛点：职业瓶颈、感情困扰、决策焦虑
- 行为：愿意为「心理安慰」付费，喜欢分享有趣体验
- Web3 契合点：匿名性 + 数据主权

**典型用户 B：国潮文化爱好者**
- 年龄：22-30 岁
- 职业：设计/文创/教育
- 痛点：市面产品「土味」，缺乏美学价值
- 行为：愿意为「美感」付费，社交平台活跃
- Web3 契合点：NFT 收藏 + 社交炫耀

**典型用户 C：海外华人**
- 年龄：25-40 岁
- 地点：北美/欧洲/澳洲
- 痛点：文化断层，需要文化连接
- 行为：愿意为「文化认同」付费，代币支付便捷
- Web3 契合点：跨境支付 + 去中心化访问

## 三、渠道通路（Channels）

### 3.1 全渠道矩阵

| 阶段 | 渠道 | 目的 | Web3/Web4 创新 |
|------|------|------|-----------------|
| **认知** | 抖音/小红书（短视频） | 病毒传播，获取流量 | Web3：空投代币吸引早期用户 |
| **兴趣** | 微信公众号/知乎 | 深度内容，建立信任 | Web4：AI Agent 自动回复私信 |
| **考虑** | 微信小程序（轻量体验） | 降低试用门槛 | Web3：连接钱包即注册 |
| **购买** | 独立 App/Web（完整功能） | 深度用户转化 | Web3：代币支付折扣 |
| **留存** | 社群（微信/Discord/Telegram） | 社区运营，提升 LTV | Web3：DAO 治理参与感 |
| **推荐** | 社交分享（卦象海报/NFT） | 裂变传播 | Web3：邀请奖励代币 |

### 3.2 渠道策略详解

**阶段 1：冷启动（0-1000 用户）**
- 抖音/小红书发布「手势摇卦」 demo 视频
- 知乎回答「六爻占卜」相关问题，植入产品
- 空投 100 $YAO 给前 1000 名注册用户

**阶段 2：增长（1000-10000 用户）**
- 微信小程序上线，降低试用门槛
- KOC 合作（国潮博主、玄学博主）
- 邀请奖励：每邀请 1 人获得 10 $YAO

**阶段 3：规模化（10000+ 用户）**
- 独立 App 上线（iOS/Android）
- Vision Pro 版本开发（Web4 先发优势）
- 企业客户拓展（B端收入）

**阶段 4：生态化（Web3 方向）**
- Lens Protocol/Farcaster 社交协议集成
- 跨链桥接（ETH/SOL/BNB Chain）
- DAO 治理上线，社区自治

- 代币持有者可以投票决定产品功能优先级

- 7×24 可用，用户随时可以问「今天运势如何」

- 多模态交互：语音对话 + 手势交互 + 眼动追踪（未来）

**情感连接设计**
- 年度卦象回顾：类似 Spotify Wrapped，生成个人年度命运报告
- 卦象时间轴：可视化展示一年来的占卜记录，形成「命运地图」
- 社交对比：匿名对比自己与同龄人的卦象分布（大数据洞察）

### 9.2 成本优化策略

**AI API 成本优化**：
- 缓存常见卦象解读（相似问题复用）
- 使用更便宜的模型（GPT-3.5）处理简单问题
- Web3：社区贡献者人工解读（代币激励，降低 API 成本）

**服务器成本优化**：
- Vercel 免费额度最大化利用
- 静态资源 IPFS 存储（去中心化，成本更低）
- Web4：边缘计算（降低延迟 + 成本）

**人力成本优化**：
- 远程办公，降低办公成本
- 代币激励绑定核心团队（降低现金薪资压力）
- 社区贡献者（代币奖励，非全职）

**营销成本优化**：
- 代币空投替代部分广告投放
- KOC 合作（产品体验 + 代币奖励，非现金）
- 病毒传播设计（邀请奖励机制）

## 十、Web3 + Web4 融合战略

### 10.1 三个阶段

**Phase 1：Web2.5（2026 Q2-Q3）**
- 中心化产品（微信小程序/独立 App）
- 代币经济叠加（用户可选择代币支付/奖励）
- 数据主权意识培养（教育用户「你的数据你做主」）

**Phase 2：Web3（2026 Q4-2027 Q1）**
- NFT 卦象功能上线
- DAO 治理启动
- 去中心化托管（IPFS/Arweave）
- 社区共治（代币持有者投票）

**Phase 3：Web4（2027 Q2+）**
- Apple Vision Pro 版本上线
- AI Agent 陪伴功能
- 空间计算体验（虚空摇卦）
- 多模态交互（语音 + 手势 + 眼动）

### 10.2 风险与应对

| 风险 | 影响 | 应对策略 |
|------|------|----------|
| 监管风险（占卜类） | 高 | 定位「文化娱乐」，避免绝对化表述 |
| 代币监管风险 | 高 | 合规为先，必要时去代币化（纯积分） |
| AI 成本上升 | 中 | 多模型备选，社区贡献替代 |
| 竞争加剧 | 中 | 技术壁垒（空间交互专利）+ 社区壁垒（DAO） |
| 技术迭代快 | 中 | 保持敏捷，持续研发投入 |
