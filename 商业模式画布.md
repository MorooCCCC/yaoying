## 一、价值主张（Value Propositions）

### 1.1 核心价值

| 价值点 | 传统竞品 | 爻影创新 |
|--------|----------|----------|
| **占卜体验** | 文字输入 + 静态结果 | 手势摇卦 + 音效 + 动画 |
| **解卦质量** | 模板化解读 | GPT-4o 流式个性化报告 |
| **美学设计** | 土味、广告多 | 新中式极简，无广告 |
| **技术亮点** | 音频文件加载慢 | 纯代码音效引擎，零文件 |

### 1.2 价值主张详细描述

**核心产品价值**：
- 「以手摇卦，以智解象」——传统六爻 + 现代 AI 的融合体验
- 新中式极简美学，区别于市面「土味」占卜 App
- 纯代码音效引擎，无音频文件的轻量化体验
- 手势识别摇卦，增强仪式感和沉浸感

**差异化优势**：
- **交互创新**：MediaPipe 手势识别，双手摇晃触发摇卦
- **音效创新**：Web Audio API 纯合成，铜钱摇晃/落定/爻成音效
- **AI 深度**：GPT-4o 流式输出，个性化解读结合用户画像
- **美学设计**：天青/朱砂/墨色配色，宋体排版，八卦元素

## 二、客户细分（Customer Segments）

### 2.1 用户分层

| 层级 | 用户画像 | 需求痛点 | 付费意愿 |
|------|----------|----------|----------|
| **C1 核心用户** | 25-35岁都市白领，本科+，对传统文化有兴趣 | 压力大、焦虑，需要情绪出口 + 决策辅助 | 高（¥10-50/月） |
| **C2 文化爱好者** | 国潮消费者，汉服/茶道/国学爱好者 | 文化认同，追求「正宗」体验 | 中（¥5-20/月） |
| **C3 海外华人** | 海外留学生/工作者，文化归属感需求 | 异国他乡，需要文化连接 + 心理安慰 | 高（$5-20/月） |
| **C4 好奇尝鲜者** | Z 世代，喜欢新鲜体验 | 社交货币，朋友圈分享 | 低（免费 + 偶尔付费） |
| **B1 企业客户** | 文创品牌、博物馆、文化机构 | 需要国风互动体验装置 | 高（¥5万-50万/项目） |
| **B2 开发者** | 需要音效/手势 SDK 的开发者 | 轻量化、无版权风险的解决方案 | 中（$20-100/月） |

### 2.2 用户画像详情

**典型用户 A：都市焦虑白领**
- 年龄：28-35 岁
- 职业：互联网/金融/咨询
- 痛点：职业瓶颈、感情困扰、决策焦虑
- 行为：愿意为「心理安慰」付费，喜欢分享有趣体验

**典型用户 B：国潮文化爱好者**
- 年龄：22-30 岁
- 职业：设计/文创/教育
- 痛点：市面产品「土味」，缺乏美学价值
- 行为：愿意为「美感」付费，社交平台活跃

**典型用户 C：海外华人**
- 年龄：25-40 岁
- 地点：北美/欧洲/澳洲
- 痛点：文化断层，需要文化连接
- 行为：愿意为「文化认同」付费

## 三、渠道通路（Channels）

### 3.1 全渠道矩阵

| 阶段 | 渠道 | 目的 |
|------|------|------|
| **认知** | 抖音/小红书（短视频） | 病毒传播，获取流量 |
| **兴趣** | 微信公众号/知乎 | 深度内容，建立信任 |
| **考虑** | 微信小程序（轻量体验） | 降低试用门槛 |
| **购买** | 独立 App/Web（完整功能） | 深度用户转化 |
| **留存** | 社群（微信群） | 社区运营，提升 LTV |
| **推荐** | 社交分享（卦象海报） | 裂变传播 |

### 3.2 渠道策略详解

**阶段 1：冷启动（0-1000 用户）**
- 抖音/小红书发布「手势摇卦」 demo 视频
- 知乎回答「六爻占卜」相关问题，植入产品
- 前 1000 名注册用户赠送免费解卦次数

**阶段 2：增长（1000-10000 用户）**
- 微信小程序上线，降低试用门槛
- KOC 合作（国潮博主、玄学博主）
- 邀请奖励：每邀请 1 人获得 1 次免费解卦

**阶段 3：规模化（10000+ 用户）**
- 独立 App 上线（iOS/Android）
- 企业客户拓展（B端收入）

## 四、客户关系（Customer Relationships）

### 4.1 关系类型

| 用户层级 | 关系类型 | 具体形式 |
|----------|----------|----------|
| C1 核心用户 | 个性化服务 | AI 深度解卦 + 年度运势报告 |
| C2 文化爱好者 | 社群运营 | 卦象解读社群 + 文化活动 |
| C3 海外华人 | 自助服务 + 社区 | 多语言 + 全球社群 |
| C4 好奇尝鲜者 | 自动化服务 | 免费基础功能 |
| B1 企业客户 | 专属服务 | 定制化解决方案 |
| B2 开发者 | 开发者关系 | 文档 + SDK + 技术支持 |

### 4.2 客户关系创新

**情感连接设计**：
- 年度卦象回顾：生成个人年度命运报告
- 卦象时间轴：可视化展示一年来的占卜记录
- 社交对比：匿名对比自己与同龄人的卦象分布

**会员权益设计**：
- 免费用户：每日 1 次基础解卦
- 会员用户：无限次解卦 + 深度报告 + 历史记录
- 企业客户：API 调用 + 定制化 + 专属客服

## 五、收入来源（Revenue Streams）

### 5.1 收入模型

| 收入流 | 定价 | 目标用户 |
|--------|------|----------|
| **会员订阅** | ¥19.9/月，¥199/年 | C1 核心用户 |
| **单次付费** | ¥3-5/次 | C2/C4 用户 |

| **企业定制** | ¥5万-50万/项目 | B1 企业客户 |
| **SDK 授权** | $20-100/月 | B2 开发者 |

### 5.2 收入预测（3 年）

**Year 1（2026）**：
- 目标用户：10,000
- 付费用户：500（5% 转化率）
- 月收入：¥10,000（会员）+ ¥3,000（单次）+ ¥2,000（企业）= ¥15,000/月
- 年收入：¥180,000

**Year 2（2027）**：
- 目标用户：100,000
- 付费用户：10,000（10% 转化率）
- 月收入：¥150,000（会员）+ ¥30,000（单次）+ ¥50,000（企业）= ¥230,000/月
- 年收入：¥2,760,000

**Year 3（2028）**：
- 目标用户：500,000
- 付费用户：75,000（15% 转化率）
- 月收入：¥1,000,000（会员 + 单次）+ ¥200,000（企业）= ¥1,200,000/月
- 年收入：¥14,400,000

## 六、核心资源（Key Resources）

### 6.1 资源清单

| 资源类型 | 具体内容 | 战略价值 |
|----------|----------|----------|
| **技术资产** | Web Audio 音效引擎、MediaPipe 手势方案、干支算法 | 高（技术壁垒） |
| **数据资产** | 64 卦数据库、用户占卜记录、AI 解卦模板 | 高（网络效应） |
| **品牌资产** | 「爻影」品牌、新中式美学设计系统 | 中（差异化） |
| **人力资产** | 开发团队、AI 训练师、社区运营 | 高（执行能力） |
| **社区资产** | 早期用户、KOC、核心粉丝 | 高（网络效应） |
| **资金资产** | 启动资金、运营资金 | 中（运营保障） |

### 6.2 资源获取策略

**技术资产**：
- 开源非核心模块（吸引开发者贡献）
- 专利申请：空间手势交互、音效合成算法
- 持续研发：保持技术领先

**数据资产**：
- 构建「卦象大数据」——谁问什么问题的趋势分析
- 用户授权后匿名化分析

**品牌资产**：
- 国潮联名（汉服品牌、茶饮品牌）
- 文化 IP 合作（博物馆、非遗项目）

## 九、成本结构（Cost Structure）

### 9.1 成本清单

| 成本类型 | 具体内容 | 月成本（Year 1） |
|----------|----------|-------------------|
| **AI API 成本** | GPT-4o 调用费用 | ¥2,000 |
| **服务器成本** | Vercel 托管、数据库 | ¥500 |
| **人力成本** | 开发/运营/设计 | ¥20,000 |
| **营销成本** | 投放/KOC 合作 | ¥5,000 |
| **法律服务** | 合规/商标注册 | ¥1,000 |
| **办公成本** | 远程办公，低成本 | ¥500 |
| **合计** | | **¥29,000/月** |

### 9.2 成本优化策略

**AI API 成本优化**：
- 缓存常见卦象解读（相似问题复用）
- 使用更便宜的模型（GPT-3.5）处理简单问题

**服务器成本优化**：
- Vercel 免费额度最大化利用
- 静态资源 CDN 加速

**人力成本优化**：
- 远程办公，降低办公成本
- 核心团队 + 兼职/外包结合

**营销成本优化**：
- 病毒传播设计（邀请奖励机制）
- KOC 合作（产品体验置换，非现金）
